L’inflation, une taxe qui ne dit pas son nom ?

Dans l’article précédent, on a montré que lorsque les salaires bruts sont revalorisés au taux de l’inflation, le pouvoir d’achat baisse car la progression du salaire net est inférieure à celle du coût de la vie après impôt.

Dans cet article je cherche à comprendre la relation entre les niveaux d’inflation et de revenu et la perte relative de pouvoir d’achat. À partir d’un modèle d’imposition allemand, je calcule la perte relative de pouvoir d’achat comme ça :

\begin{equation*} \text{perte relative de pouvoir d'achat} = \frac{\text{revenu net après inflation}}{\text{revenu net avant inflation} \times (1 + \text{inflation})} - 1 \end{equation*}

Comme l’inflation s’applique en moyenne à tous les biens et services, je multiplie le revenu net avant inflation (qui me sert de base de pouvoir d’achat) par 1 plus l’inflation pour obtenir l’équivalent après inflation. C’est ce qu’il faudrait gagner en net pour garder le pouvoir d’achat constant.

Par exemple si avant inflation j’ai gagné 20k et que l’inflation est de 10%, il faudrait que je gagne 22k net après inflation. Si je gagne seulement 21.5k net après inflation, j’ai perdu 500 balles soit 2.27% de pouvoir d’achat.

Le mieux est en vert, le pire en rouge. Dans lisser ses revenus, le premier graphique montre le taux marginal d’imposition. On peut faire le lien entre l’augmentation rapide du taux marginal jusqu’à 58k de revenu annuel et la tranche des revenus qui pert le plus de pouvoir d’achat relatif : 55k.

Autrement dit, la perte relative de pouvoir d’achat augmente avec l’inflation (plus d’inflation = plus de perte de pouvoir d’achat relatif même avec un ajustement des salaires bruts) et augmente avec le niveau de revenu jusqu’à environ 60k, après quoi elle commence à baisser pour se stabiliser.

Les revenus très bas sont peu impactés (toujours en supposant que leurs revenus suivent l’inflation, ce qui n’est généralement pas le cas mais c’est un autre sujet) car ils paient peu d’impôt sur le revenu. Pour les plus hauts revenus, ils perdent aussi du pouvoir d’achat mais cette perte tend à se stabiliser.

Qui est le grand gagnant du jeu de la lose ? La classe moyenne. À taux d’inflation égal, elle est plus impactée que les pauvres et les riches.

Un ménage avec 50k de revenu brut perd 1.12% de pouvoir d’achat quand l’inflation est de 5% et c’est encore en supposant que son revenu brut a été ajusté à la véritable inflation. En 2021 alors que le prix d’à peu près tout augmente dans des proportions dignes des années 1970 aux États-Unis, le montant du SMIC a progressé de 0.9% en 2022.

Classe moyenne : à vos marques, prêts, perdez !

Voilà le code qui permet de reproduire les calculs.

from dataclasses import dataclass
from typing import Dict, List

import pandas as pd
import numpy as np


class TaxRate:
    def calculate_taxes(self, income: float) -> float:
        raise NotImplementedError

    def calculate_marginal_rate(self, income: float) -> float:
        raise NotImplementedError

    def calculate_effective_rate(self, income: float) -> float:
        raise NotImplementedError


@dataclass
class ProportionalTaxRate(TaxRate):
    rate: float

    def calculate_taxes(self, income: float) -> float:
        return self.rate * income

    def calculate_marginal_rate(self, income: float) -> float:
        return self.rate

    def calculate_effective_rate(self, income: float) -> float:
        return self.rate


@dataclass
class ProgressiveTaxRate(TaxRate):
    min_rate: float
    max_rate: float
    max_amount: float

    def calculate_taxes(self, income: float) -> float:
        marginal_rate = self.calculate_marginal_rate(income)
        effective_rate = (self.min_rate + marginal_rate) / 2
        return income * effective_rate

    def calculate_marginal_rate(self, income: float) -> float:
        return self.min_rate + (self.max_rate - self.min_rate) * (income / self.max_amount)

    def calculate_effective_rate(self, income: float) -> float:
        marginal_rate = self.calculate_marginal_rate(income)
        return (self.min_rate + marginal_rate) / 2


@dataclass
class TaxBracket:
    from_income: float
    to_income: float
    tax_rate: TaxRate

    def _calculate_income_in_bracket(self, income: float) -> float:
        if income < self.from_income:
            return 0
        else:
            return min(income, self.to_income) - self.from_income

    def calculate_taxes(self, income: float) -> float:
        bracket_income = self._calculate_income_in_bracket(income)
        return self.tax_rate.calculate_taxes(bracket_income)

    def calculate_marginal_rate(self, income: float) -> float:
        bracket_income = self._calculate_income_in_bracket(income)
        if bracket_income == 0:
            return 0
        return self.tax_rate.calculate_marginal_rate(bracket_income)


class GermanIncomeTaxCalculator:
    """
    tax_engine = GermanIncomeTaxCalculator(
        tax_brackets=[
            TaxBracket(0, upper_bracket1, ProportionalTaxRate(0)),
            TaxBracket(upper_bracket1, upper_bracket2,
                ProgressiveTaxRate(0.14, 0.42, upper_bracket2 - upper_bracket1)
            ),
            TaxBracket(upper_bracket2, upper_bracket3, ProportionalTaxRate(0.42)),
            TaxBracket(upper_bracket3, np.inf, ProportionalTaxRate(0.45))
        ]
    )
    assert tax_engine.compute_taxes(399_000, 0.055, 0) == 169_300.914
    """
    def __init__(
        self,
        tax_brackets: List[TaxBracket],
        solidarity_tax_rate: float,
        church_tax_rate: float
    ) -> None:
        self.tax_brackets = tax_brackets
        self.solidarity_tax_rate = solidarity_tax_rate
        self.church_tax_rate = church_tax_rate

    def calculate_taxes(self, income: float) -> float:
        self.income_tax = 0
        for tax_bracket in self.tax_brackets:
            self.income_tax += tax_bracket.calculate_taxes(income)
        self.solidarity_tax = self.income_tax * self.solidarity_tax_rate
        self.church_tax = self.income_tax * self.church_tax_rate
        return self.income_tax + self.solidarity_tax + self.church_tax

    def calculate_rates(self, income: float) -> Dict[str, float]:
        marginal_rate = max(tb.calculate_marginal_rate(income) for tb in self.tax_brackets)
        if income == 0:
            effective_rate = 0
        else:
            effective_rate = self.calculate_taxes(income) / income
        return {"marginal_tax_rate": marginal_rate, "effective_tax_rate": effective_rate}



def main():
    upper_bracket1 = 9_744
    upper_bracket2 = 57_918
    upper_bracket3 = 274_612

    tax_engine = GermanIncomeTaxCalculator(
        tax_brackets=[
            TaxBracket(0, upper_bracket1, ProportionalTaxRate(0)),
            TaxBracket(upper_bracket1, upper_bracket2,
                ProgressiveTaxRate(0.14, 0.42, upper_bracket2 - upper_bracket1)
            ),
            TaxBracket(upper_bracket2, upper_bracket3, ProportionalTaxRate(0.42)),
            TaxBracket(upper_bracket3, np.inf, ProportionalTaxRate(0.45))
        ],
        solidarity_tax_rate=0.055,
        church_tax_rate=0
    )

    annual_incomes = np.arange(5_000, 105000, 5_000)
    inflation_rates = np.arange(0, 0.16, 0.01)

    data = []
    for annual_income in annual_incomes:
        tax_amount_before = tax_engine.calculate_taxes(income=annual_income)
        net_income_before = annual_income - tax_amount_before
        for inflation_rate in inflation_rates:
            annual_income_after = (1 + inflation_rate) * annual_income
            tax_amount_after = tax_engine.calculate_taxes(income=annual_income_after)
            net_income_after = annual_income_after - tax_amount_after
            pct_change_in_col = net_income_after / (net_income_before * (1 + inflation_rate)) - 1
            data.append([annual_income, inflation_rate, pct_change_in_col * 100])

    data = pd.DataFrame(
        data,
        columns=("annual_income", "inflation_rate", "loss_in_purchasing_power")
    )


if __name__ == "__main__":
    main()
